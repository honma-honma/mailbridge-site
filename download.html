ダウンロードとインストール手順

はじめての方でも迷わないよう、順番に画面でご説明します。
所要時間は約5分。難しい操作はありません。

ご利用には Windows 10／11 のパソコンが必要です。macOS（Mac）・スマートフォン・タブレットには対応していません。

1

下のボタンから「ダウンロードページ」を開く

このページの下にある青いボタンを押すと、配布元（GitHub）のダウンロードページが新しいタブで開きます。GitHubは世界中で使われている、安全なソフト配布サービスです。

2

「Assets（ファイル一覧）」の中から .exe を押す

開いたページの下のほうに「Assets」という、ファイルが並んだ一覧があります。その中の「.exe」で終わるファイル（下の図の緑色の行）をクリックすると、ダウンロードが始まります。

Assets
MailBridge-Setup-x.x.x.exe これを押す
Source code (zip) ← 押さなくてOK
Source code (tar.gz) ← 押さなくてOK

※ファイル名のバージョン番号（x.x.x の部分）は時期によって変わります。末尾が「.exe」のものを選べば大丈夫です。「Source code」は開発者向けなので押す必要はありません。

3

ダウンロードした .exe をダブルクリックして開く

画面の下や「ダウンロード」フォルダに保存された MailBridge-Setup-….exe をダブルクリック。あとは画面の案内に沿って進めるだけでインストールが完了します。

4

青い警告が出たら「詳細情報」→「実行」

起動時に「Windows によって PC が保護されました」という青い画面が出ることがあります。これは、まだ署名登録をしていない新しいアプリで表示されるもので、故障やウイルスではありません。

操作のしかた

① 「詳細情報」という小さな文字をクリック
② 下に出てくる「実行」ボタンを押す

※どうしても不安な場合は、無理に進めず下記の連絡先までお問い合わせください。

ダウンロードページを開く（GitHub）

新しいタブで開きます。上の手順2にしたがって「.exe」を選んでください。

うまくいかない・不安なときは

ダウンロードやインストールでお困りの場合は、お気軽にご連絡ください。画面の状態を教えていただければサポートします。
