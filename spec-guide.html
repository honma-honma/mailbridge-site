PC SPEC GUIDE

PCスペックの見方

無料のローカルAIが動くか確認するために、お使いのPCの性能を調べます。むずかしくありません。確認するのは3つだけです。

START

確認するのは3つだけです

専門知識はいりません。Windowsの「タスクマネージャー」という標準の画面で、次の3つを見るだけです。メモして、シミュレーターに入力すれば判定できます。

🎮
① VRAM
GPUのメモリ
🧠
② RAM
メインメモリ
⚙️
③ CPU / GPU
頭脳と計算担当
タスクマネージャーの開き方：キーボードの Ctrl + Shift + Esc を同時に押す → 上の「パフォーマンス」タブを開きます。（または、画面下のタスクバーを右クリック →「タスク マネージャー」）

※説明用イラスト（実際の画面に近づけたイメージです）

🎮

VRAM（GPU搭載メモリ）

  • ●AIモデルの処理に使う、GPU専用のメモリです。
  • ●多いほど大きな（賢い）モデルが動きます。ローカルAIではいちばん重要です。
確認のしかた：タスクマネージャー →「パフォーマンス」→ 左の 「GPU」 を選択 → 「専用GPUメモリ」の数字（例：8.0 GB）を見ます。

※説明用イラスト（実際の画面に近づけたイメージです）

🧠

RAM（メインメモリ）

  • ●PC全体の「作業机の広さ」です。広いほど一度にたくさんの作業ができます。
  • ●VRAMとは別物です（GPUがないPCでは、このRAMでAIを動かします）。
確認のしかた：タスクマネージャー →「パフォーマンス」→ 左の 「メモリ」 を選択 → 右上の合計（例：16.0 GB）を見ます。

※説明用イラスト（実際の画面に近づけたイメージです）

⚙️

CPU

  • ●PCの「頭脳」です。新しい世代ほど速い傾向があります。
  • ●GPUがないPCでローカルAIを動かすときは、ここの性能が速度に効きます。
確認のしかた：タスクマネージャー →「パフォーマンス」→ 左の 「CPU」 を選択 → 右上に名前（例：Intel Core i5-12400）が出ます。

※説明用イラスト（実際の画面に近づけたイメージです）

🖥️

GPU

  • ●AIの計算を担当するパーツです。あると、ローカルAIが大きく速くなります。
  • ●名前に 「RTX」「GTX」（NVIDIA製）があれば、高性能GPU搭載です。
確認のしかた：タスクマネージャー →「パフォーマンス」→ 左の 「GPU」 を選択 → 右上の名前（例：NVIDIA GeForce RTX 4060）を見ます。「Intel」「AMD Radeon (Graphics)」のみの場合は内蔵グラフィックスで、GPUなし相当です。

※説明用イラスト（実際の画面に近づけたイメージです）

確認できたら、判定してみましょう
